Utilitaire PDF Combiner

Cet utilitaire est un projet personnel réalisé juste après mon apprentissage du langage python. J'ai utilisé le framework Kivy pour l'interface utilisateur et le module PyPDF2 de Python pour manipuler les PDF.

J'ai volontairement fait en sorte que l'interface soit simple avec uniquement les fonctionnalités essentielles pour combiner des pdfs, afin que le plus grand nombre puisse l'utiliser facilement. L'utilisateur doit glisser/déposer les fichiers PDFs qu'il souhaite fusionner dans la fenêtre du programme et cliquer sur combiner. Une interface de type "enregistrer sous" implémentée grâce au module Tkinter s'ouvre alors pour choisir le répertoire et le fichier de sortie.

Les cas particuliers sont gérés grâce à un bandeau coloré d'avertissement ou d'erreur qui apparait en cas de besoin.


Technologies utilisées
:
Python, Kivy, PyPDF2, Tkinter
